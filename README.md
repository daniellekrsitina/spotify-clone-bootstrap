🎵 Clone da Página Inicial do Spotify

Este projeto é uma recriação da página inicial do Spotify, desenvolvida 100% com HTML, CSS e Bootstrap.  
O objetivo é praticar conceitos de responsividade, grid, componentes prontos e personalização de estilos, acompanhando aulas e aplicando na prática.

✨ Funcionalidades
- Estrutura completa da página inicial do Spotify  
- Uso exclusivo do framework Bootstrap 
- Layout responsivo para diferentes dispositivos  
- Exercício de aprendizado e prática de front-end  

🚀 Tecnologias utilizadas
- HTML5  
- CSS3  
- Bootstrap 5  

📂 Como executar
1. Clone este repositório:
   git clone https://github.com/DanielleKrsitina/spotify-clone-bootstrap.git
